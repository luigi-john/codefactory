What is web app?

Web-based applications (web apps) are software that are purely based on the web (ie. Google Docs). They function like traditional desktop software (ie. Microsoft Word), but are accessed by web browsers.

Most people use web apps everyday without knowing it, for example: Gmail, Dropbox, iCloud, Google Drive, etc. These are websites that let people do “things”: editing documents, video chat, real-time financial information, data dashboard, location tracking, etc.
